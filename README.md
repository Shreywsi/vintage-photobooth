# Vintage Photobooth

Retro photobooth built with React + Vite.

Getting started

1. Install:

```bash
cd vintage-photobooth
npm install
npm run dev
```

2. Open the dev URL, allow camera access.

Features

- Live camera preview in a vintage frame
- Countdown and automatic capture of 4 photos
- Vertical photostrip generation and download
- Vintage filters: sepia, black & white, faded film, grain
- Flash animation and retake
